/cards/

GET

Description

Retrieves all cards within a square box of the specified length centered at the given coordinates.

Required parameters

lat

The latitude coordinate

long

The longitude coordinate

Optional parameters

length (default=100)

The dimension of the square box centered on the specified coordinates

Example

GET /cards?length=100&lat=47.6097&long=122.3331

{
  "title": "The Stumbling Monk",
  "lat": 47.6097,
  "long": 122.3331
}
